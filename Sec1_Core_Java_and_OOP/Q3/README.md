# Question 3: Latest Revision

## Overview

Latest Part Revision Finder
This repository contains a Java solution to process a list of part revisions and extract the latest revision for each unique part.

For example, given the input ["P001-A", "P001-B", "P001-C", "P002-A", "P002-B", "P003-A"], the program will correctly identify and output:

P001 - C

P002 - B

P003 - A

## Project Structure

```text
src/
└── LatestRevision.java
```

---

## How to Run

### Compile

```bash
javac src/LatestRevision.java
```

### Run

```bash
java src/LatestRevision
```
